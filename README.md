# Black Hole Simulation (BlackHoleSim)

This repository contains a Windows build archive for the Black Hole Simulation application.

- Archive: `BlackHoleSim-Windows.zip` (placed in the repository root)
- Ensure a recent JDK (Java 11 or newer) is installed if running or building from source.

Controls:
use mouse to click and drag to create particles,the more u drag,the more velocity the particle has,it's that easy
